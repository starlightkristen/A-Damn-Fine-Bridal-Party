üë• Guest List Submission Options

There are two ways to submit guest information for the bridal party:

Option 1: GitHub Issue Form

Submit guest details directly through our GitHub issue tracker. Best for organizers with GitHub accounts.

  • Navigate to repository Issues
  • Select "Guest List Submission" template
  • Fill in guest details in the format provided
  • Creates tracked record for review
Submit via GitHub

Option 2: Google Form

Alternative option for guests and organizers who prefer a simple web form interface.

  • Fill out a user-friendly Google Form
  • Responses collected in Google Sheets
  • Easy to share with multiple people
  • Export as JSON for data/guests.json
